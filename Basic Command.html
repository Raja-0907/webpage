Basic Commands

man Command

man ls: This command is used to display the documentation on the ‘ls’ command in linux

Syntax: man [COMMAND NAME]

cd Command

cd : It is used to change the current working directory to a specified folder in inside the terminal

Syntax: cd [Directory]

cd .. This command is used to navigate to parent directory of current directory

ls : This command is used to display a directory’s files and folders

ls -l : This is used to display a detailed version of every file and directory available inside the current working directory

cat Command

cat[filename] : Used to display the contents of the file as named

cat -n : Used to display the contents of the file with line number

touch Command

touch: This is used to create a new file without any content inside it

touch [filename] : Used to create a new file with the given name

touch : Used to create multiple files simultaneously

mkdir Command

Mkdir: This is used to create new directories inside an existing working directory

Syntax: Mkdir [option] .. ..

mkdir --help : used to display relevant information pertaining to the mkdir command

mkdir [filename] : used to create a folder with a specified name

pwd

pwd: This is used to display the path of the current working directory your in

Syntax:pwd -L : it displays the symbolic version of the existing path

pwd-P : It displays the actual existing path

echo

echo : This is used to display a line of text/string which is passed in as an argument . it is commonly used for debugging shell programs inside th terminal

Syntax: echo “String” : Used to display the string passed as an argument inside quotes

echo -e “learn\nBy\nDoing” : enables the echo command to recognize the backslash escape sequences inside the arguments

rm Command

rm : This is used to delete references to object from filesystem(a file with two different names)

Syntax : rm : used to delete the corresponding existing file from inside the working directory

rm -i : enables the rm command to request confirmation from the user before deleting the corresponding file

rmdir Command

rmdir : This command allows you to delete empty directories, if a directory contains files/folders inside it ,it will display an error

rmdir : used to delete any directory provided, if it is empty

wget Command

wget : It is used to download applications and web pages directly from the web

Syntax : wget : used to download applications and web pages directly from the web

wget -O : used to retry a file download for a given number of times

mv Command

mv : You can rename a file/directory You can easily move a file/directory from one location to another

Syntax: mv [Source] [Destination]

mv : used to rename a file to a different name as specified

mv : used to transfer a file from a given directory to a different directory

cp Command

cp : It is used to copy files/directory from one location to another from inside the terminal

Syntax : cp : used to copy the contents of the source file to the destination file

tree command

tree : It is used to list out the contents of directory in a tree-like structure

Syntax : tree : used to display the contents of directory as an indented tree

tree -f : used to display the full path of each working directory and file inside your current directory

grep Command

grep : It is used to search through a specified file and print sll lines that match a given pattern

Syntax : grep [pattern] [filename]

grep -i [pattern] [filename] : used to search for given pattern inside a given file . the i-tag is used to disable case sensitivity while searching for pattern

grep -c [pattern][filename]: used to display the count of the number of occurrences of a given pattern inside a specified file
