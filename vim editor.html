Vim Editor

How to install Vim Editor:

Step 1: sudo apt-get update

Step 2: sudo apt-get install vim -y

Step 3: now, vim editor is installed , you can check in the show application

Vim Commands:

  • vi : It allows the user to edit any text content inside the vim text editor from the terminal

  • Syntax : vi / vim [filename] : used to create a new file and enter it into vim editor

  • i : used to enter insert mode

  • yy : copy a line/lines inside the editor

  • p/P : paste command

  • u : undo command

  • dd : delete the line [where your cursor is]

  • ctrl + r : Redo command

  • w : save a file

  • x : save and exit the file

  • :q : Quit vim editor without saving a file

  • :q! : it will quit the file and disregard all changes that are currently in file

  • :[number]: the line you want to go in the file

  • /[line] : it will search the line, if it is not what you are searching PRESS ‘N’ it will search bottom to top

  • Shift + n : it will scroll you back to the previous item

  • gg : take you to top of the any file
