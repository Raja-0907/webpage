Shh Key

What is Shh key?

  • Ssh : secure shell management

  • It is a pair of public and private keys that are used to authenticate and establish an encrypted communication channel between client and a remote machine over the internet

    • To make an ssh connection :

    Type:ssh username@serverhost

  • Serverhost : it will be a ip address or domain name

  • Username : the user name will be the user on the remote server that your connecting not user on your local computer

    • Remote server :

    To connect to the remote server which can either be a password or a key

    There are two types of keys :

  • Private key

  • Public Key

    • Private key :

    Private key will be on your local computer

    Public key :

    Public key is provided to other resources such as remote server you to connect to and verify that the connection you create are from you

    How to create a ssh key :

    Step 1: To create a key pair on the client machine [usually your computer [ ssh-keygen ]

    Step 2: Give name for your key

    Step 3: Give password for your key

    Step 4: Give the command ‘ls’ to see your keys

    Step 5: To see your key clearly give the command cat [keyname]
